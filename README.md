# Driver Drowsiness Detection System 🚗💤

This project is a React-based web application that detects driver drowsiness using webcam input.
Driver drowsiness is a major cause of road accidents, contributing to 40% of crashes in the U.S., with over 100,000 accidents and 1,550 deaths annually. In developing countries, long drives, irregular schedules, and poor rest options worsen the problem. Drowsiness develops gradually, and even brief microsleeps (2–3 seconds) can be dangerous. 
Current solutions like EEG headsets, infrared cameras, or backend Python systems are either expensive, intrusive, or require internet/server connections. 
Proposed Solution: A real-time browser-based system using ReactJS, face-api.js, and the Web Speech API. It uses a standard laptop webcam to monitor four drowsiness signals - EAR, MAR, PERCLOS, and head nod  and provides escalating voice/audio alerts without any backend or extra hardware. 

## 🔍 Features
- Real-time face & eye detection
- Drowsiness alert system
- Simple and interactive UI
